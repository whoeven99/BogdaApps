# web-template（Vite + React）

前端页面模板：Vite + React + TypeScript。

## 启动

在 `web-template` 目录下：

- 安装依赖：`npm i`
- 开发启动：`npm run dev`
- 构建：`npm run build`
- 本地预览构建产物：`npm run preview`

## 目录结构

- `index.html`：入口 HTML
- `vite.config.ts`：Vite 配置
- `src/main.tsx`：React 入口
- `src/App.tsx`：示例页面
- `src/styles.css`：基础样式

修改 `src/App.tsx` 即可开始开发。
